Class: IntegratorIAS15

IntegratorIAS15

new IntegratorIAS15(system)

Creates an instance of IntegratorIAS15. The IntegratorIAS15 advances (in time) a System which holds a collection of Bodys This is a js port of the C version of the IAS15 integrator described in http://arxiv.org/abs/1409.4779 and available under GPL3 at https://github.com/hannorein/rebound.
Parameters:
Name Type Description
system System the system this integrator is in charge of integrating in time
