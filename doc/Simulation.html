Class: Simulation

Simulation

new Simulation(system)

Creates an instance of a Simulation. The Simulation is a wrapper for everything we need to display the time evolution; it is in charge of setting up the plots to display the state of the system, it holds a reference to a System, which holds a collection of Bodys. It also instantiates an IntegratorIAS15 to advance the system in time. The driver script calls the Simulation's methods to control the flow of the animation, including speeding up or slowing down the animation, zooming in and out, selecting and applying initial conditions, and stepping through to each subsequent animation frame.
